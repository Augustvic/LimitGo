# LimitGo

LimitGo is a set of Go libraries that includes collections and utils for concurrency, math, current-limiting, caching, and more.

## The following has been proposed:

### collections

#### list
    
   * arraylist, linkedlist
    
#### map

   * equals
   
#### queue

#### stack

#### set

### concurrency

### math

#### compute
   
   * lcm, gcd, bitcount
   
#### array util
   
#### string util
   
#### sort

   * quick sort

### current-Limiting

   * permit limiter
   * leaky limiter
   * sliding window

### caching

   * LRU
   * LFU
   * LRU2
   
### other

#### util
