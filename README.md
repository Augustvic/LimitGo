# LimitGo

LimitGo is a set of Go Libraries that includes collections and utils for concurrency, math, current-limiting, caching and more.

## The following has been proposed:

* Collections

* Concurrency

   - blocking queue

* Math

   - lcm and gcd
   - array util
   - string util
   - sort

* Current-Limiting

   - permit limiter
   - leaky limiter
   - sliding window

* Caching

   - LRU
   - LFU
   - LRU2
