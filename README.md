# LimitGo

LimitGo is a set of Go libraries that includes collections and utils for concurrency, math, current-limiting, caching, and more.

## The following has been proposed:

### collections

#### linear structure
    
   * arraylist, linkedlist, queue, stack, deque
   
#### map

   * skiplistmap
   
#### set

   * set, sortedset

### math, string, array

### current-Limiting

   * permit limiter
   * leaky limiter
   * sliding window

### caching

   * LRU
   * LFU
   * LRU2
